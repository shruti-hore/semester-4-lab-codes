# Programming Logic with Java Lab

## About
This repository contains Java programs developed as part of laboratory experiments to understand core Java concepts such as object-oriented programming, exception handling, collections, file handling, JDBC, and GUI development.

## List of Experiments
1. Menu-Driven Calculator  
2. Vehicle Management System  
3. Book Inventory System  
4. Vector Operations Program  
5. Banking Application  
6. Employee Payroll System  
7. Student Record Manager  
8. Collections & Design Patterns  
9. JDBC Database Application  
10. JavaFX GUI Application
